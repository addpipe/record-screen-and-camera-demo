# getDisplayMedia() With Camera Demo

This demo uses `getDisplayMedia()`, `HTML Canvas` and the `Media Stream Recording API` to record the screen and the camera at the same time.

Works on:
* Chrome 74 and up
* Edge 79 and up
* Firefox 66 and up
* Opera 60 and up
